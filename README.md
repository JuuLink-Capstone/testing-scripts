# Juulink Testing Scripts

This repository contains scripts to assist in testing components of the Juulink capstone project. 

* [failover](./failover) contains scripts to automate timed failover, as well as testing automation scripts.
* [locust](./locust) contains scripts to run locust.
* [doc](./doc) contains extra documentation about specific topics.
  * [doc/copilot-agent-permissions.md](./doc/copilot-agent-permissions.md) describes what permissions the GitHub Copilot coding agent has in this repository.
